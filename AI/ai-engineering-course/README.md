# AI Engineering Mastery Course

> **Learn by building.** A complete AI engineering education system delivered through the NexaAI Studio roleplay simulation — 16 projects across 4 tiers, from internal tools to portfolio-level production systems.

## What This Is

This is not a traditional course. It's a **simulated AI consultancy** where every project is a real client need with business stakes, deadlines, and a CTO who won't approve anything that wouldn't survive in production.

**You learn by doing.** Each project follows a 5-phase cycle: Brief (business context) → Learning Path (just-in-time instruction) → Build (hands-on production) → Review (CTO quality bar) → Debrief (consolidation + forward chaining).

## How to Use

1. Read `system.md` to understand the NexaAI Studio simulation
2. Review `01-syllabus/ai-engineering-syllabus.md` to understand the full roadmap
3. Read `02-teaching-methodology/nexaai-teaching-methodology.md` to understand how you'll learn
4. Start with `04-course-content/tier-1-foundations/` — each project builds on the last
5. Paste `system.md` as the system prompt in a new Claude conversation to begin
