# Phase 0: AI Engineering Mindset

**Goal**: Before writing a single line of AI code, learn how senior AI engineers think — about systems, evaluation, cost, and tradeoffs.

**Time**: ~3 hours (read + reflection)

## Why This Phase Exists Before Everything Else

Framework syntax expires every 6 months. APIs change. Models get cheaper. But the way senior engineers THINK about AI systems stays constant. This phase installs the mental models you'll use across every subsequent phase.

## What You'll Learn

1. **Senior vs Junior Mindset** — The exact thought patterns that separate $70K from $250K+
2. **Production vs Academic Thinking** — Why benchmark scores lie and production metrics matter
3. **Evaluation-First Philosophy** — If you can't measure it, you can't ship it
4. **The Build Loop** — Problem → Raw → Framework → Productionize → Evaluate → Debug
5. **Cost-Awareness Culture** — Every token costs money; design accordingly
6. **Decision Framework** — When to build from scratch, when to use a framework, when to buy
