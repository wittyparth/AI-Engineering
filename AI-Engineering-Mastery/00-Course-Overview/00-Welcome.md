# Welcome to AI Engineering Mastery

**Goal**: Install prerequisites, set up your environment, and understand how this course works.

## Before You Start

**STOP.** Read Phase 0 first (`00-AI-Engineering-Mindset/`). It installs the mental models you'll use throughout this course. Don't skip it.

## What This Course Is

This course teaches you to BUILD production AI systems — the kind that companies deploy for real users. You will:

1. **Build from scratch first** — Understand every layer before abstracting
2. **Add frameworks** — Use LangChain, LlamaIndex, Pydantic AI, LangGraph, LiteLLM, DSPy — but only after you understand the problem they solve
3. **Productionize** — Add monitoring, cost controls, security, scaling
4. **Evaluate** — Measure quality, catch regressions, prove it works
5. **Debug** — Find and fix the common failure modes

## What This Course Is NOT

- NOT a tutorial ("follow along and type this code")
- NOT framework-focused ("learn LangChain syntax")
- NOT theory-only ("understand transformers math")
- NOT toy projects ("build a chatbot")

## Course Flow

```
Read Phase 0 → Setup Environment → Phase 1 → Phase 2 → ... → Capstone
```

## Your Commitment

This is a senior-level course. Each phase requires 8-18 hours. You should:
- Write code, don't just read
- Build the projects, don't just understand the concepts
- Complete the gate checks before moving on
- Keep a portfolio log
- Debug before asking for help
