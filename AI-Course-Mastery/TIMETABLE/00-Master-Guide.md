# 🗓️ AI Course Mastery — Challenge Timetable

## Your Contract

1. **6 days on, 1 day off.** Every 6 study days = 1 rest day. No exceptions.
2. **Type every line of code.** No copy-paste. Break it. Fix it. Own it.
3. **Do the flashback.** 5 minutes at end of every day. Non-negotiable.
4. **Gate checks are walls.** Do not pass a gate until you've proven competence.
5. **Log or it didn't happen.** Fill your progress log daily.

---

## Daily Session Template

```
☀️ MORNING (2.5-3h) — Deep Focus Mode
     Active recall warmup (5 min): recall yesterday's key concept from memory
     Read + annotate the assigned file
     Type out key code examples (not copy-paste!)
     Write 3-5 bullet summary in your own words

🌤️ AFTERNOON (2.5-3h) — Build Mode
     Complete the drill / exercise / project section
     Or: continue reading if the concept file is large
     Apply: modify the code, introduce a bug, then fix it

🌙 EVENING (0-1h, optional) — Review Mode
     Daily Flashback: close all files, write down 3 things you learned today
     Update your progress log
     Preview tomorrow's content
```

**Total effective learning: ~5-6 hours/day.** Don't try to do more. Diminishing returns kicks in hard after 6 hours of deep cognitive work.

## Revision System — 5 Layers

### Layer 1: Daily Flashback (🌙 Evening, 5 min)
Close everything. On paper or a file, write 3 things you learned today from memory.
This forces your brain to consolidate. **Do not skip this.**

### Layer 2: Weekly Review (Every 6th study day)
One FULL day dedicated to reviewing the past 6 days. Protocol:
1. **Scan all files** from the week — re-read headings, code snippets, failure modes
2. **Re-type key code** from 2 drills/exercises (without looking at original)
3. **Write a one-page summary** of the week's concepts from memory
4. **Identify weak areas** — note what you're still fuzzy on
5. **Gate Check** — if you can't pass it, redo before advancing

### Layer 3: Phase-End Review (After each phase)
One FULL day after completing a phase. Protocol:
1. **Mind map** the entire phase from memory on a blank page
2. **Re-do the phase Gate Check** (if any section fails, re-study that section)
3. **Write a "teaching document"** — explain the phase as if teaching someone else
4. **Update your phase checklist** in the progress log
5. **Rate your confidence** (1-5) on each topic from the phase

### Layer 4: Milestone Review (After Phase 5 — Midpoint)
3 FULL days. Cumulative review of ALL content from Phases 0-5.
This is the most important review in the entire course. By now you have a solid foundation,
and this review cements it permanently.
- Day 1: Phases 0-1 review + self-test
- Day 2: Phases 2-3 review + self-test
- Day 3: Phases 4-5 review + self-test + weak area remediation

### Layer 5: Final Sweep (After Phase 10, Before Capstone)
5 FULL days reviewing ALL 10 phases before you start the Capstone.
- Day 1: Phases 0-2
- Day 2: Phases 3-4
- Day 3: Phases 5-6
- Day 4: Phases 7-8
- Day 5: Phases 9-10 + full self-test battery

## Self-Test System

Every phase has a **Self-Test Day** built into its Phase-End Review with these question types:

| Type | Description |
|------|-------------|
| **Recall** | "What is X? Explain without notes." |
| **Apply** | "Given scenario Y, which approach would you use and why?" |
| **Compare** | "Compare A vs B — when would you choose each?" |
| **Debug** | "This code is broken. Find and fix the 3 issues." |
| **Design** | "Design a system that does Z. What are your tradeoffs?" |

The specific self-test questions are in [05-Self-Tests.md](./05-Self-Tests.md).

## How To Use This Timetable

### Each day you should:
1. Open the phase file for your current day
2. Follow the ☀️ morning → 🌤️ afternoon → 🌙 evening structure
3. Check off items in the daily checklist
4. Update your progress log at day end

### Tracking your pace:
- **Behind by 1-2 days?** Skip optional evening sessions to catch up. Do NOT skip review days.
- **Ahead by 3+ days?** Spend extra time on drills or start pre-reading the next phase.
- **Stuck on a concept?** Spend no more than 1 extra day. Move on and come back at next review.

### If you miss a day:
- Miss 1 day → pick up where you left off (skip optional evening sessions to make up time)
- Miss 2-3 days → do a 30-min warmup review of the week before resuming new content
- Miss 5+ days → spend a full review day before moving forward
