# 🔄 Review System — All Review Days

This file contains ALL review days across the entire course:
- **Weekly reviews** (every 6 study days)
- **Phase-end reviews** (after each phase)
- **Milestone review** (after Phase 5 — cumulative Phases 0-5)
- **Final sweep** (after Phase 10, before Capstone — cumulative Phases 0-10)

**When to use this file:**
- When a phase file says "Phase X Review — see Review-System.md"
- Every 6th study day for weekly review
- After Phase 5 for the Milestone Review
- After Phase 10 for the Final Sweep
- Before Capstone Day 1

---

## Review Day Protocol (Use This Every Time)

All review days follow this base protocol:

```
☀️ MORNING (2.5-3h) — Scan + Active Recall
   1. Scan section headings of all files being reviewed (5-10 min each)
   2. Close everything
   3. Write everything you remember from EACH file from memory
   4. Check what you missed — those are your weak spots

🌤️ AFTERNOON (2.5-3h) — Deep Work on Weak Spots
   1. Re-read sections you were fuzzy on
   2. Re-type important code from memory (at least 2 code examples)
   3. Re-do 1 drill/exercise from the phase
   4. Complete the self-test

🌙 EVENING (0-1h) — Reflect + Log
   1. Rate your retention (1-5) per topic
   2. Update progress log
   3. Decide: are you ready to advance or need another day?
```
