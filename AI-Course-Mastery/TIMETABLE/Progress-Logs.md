# 📊 Progress Logs — Templates & Tracking

This file contains templates for tracking your progress throughout the course.

**How to use:** Copy the relevant template into your own notes (or create files in this directory) and fill them in each day/week/phase.

---

## 1. Daily Log Template

Create a copy per day. One file per phase works well (e.g., `Phase1-Daily-Logs.md`).
