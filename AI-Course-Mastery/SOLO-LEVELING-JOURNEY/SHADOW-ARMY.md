# 👻 SHADOW ARMY — All Extracted Concepts

```
[SYSTEM]─────────────────────────────────────
  The Shadow Army contains every concept
  you've mastered. Each soldier is a concept
  you extracted — explained in your own words.
  
  When you need to remember something,
  summon its shadow. It will serve you.
─────────────────────────────────────────────
```

---

## How To Use This

1. **After every dungeon**, extract the shadow — write the concept in your own words
2. **Every morning in your Daily Quest**, summon 2 random shadows — re-read them
3. **When revising**, cover the right column and try to explain from memory
4. **When you can explain without looking**, that soldier is battle-ready
