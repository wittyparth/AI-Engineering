## The Three Phases of Every Day

Every day follows the same three-phase structure. Once this becomes automatic, your brain stops wasting energy on "what should I do?" and focuses entirely on the work.

```
☀️ DAWN   (15 min) — Daily Quest Prep
🌤️ DAY    (180 min) — Dungeon Raid (2 × 90-min sprints)
🌙 DUSK   (10 min)  — Shadow Extraction + Gate Check

Total: ~3.5 hours of focused work
```

---

## ☀️ PHASE 1: DAWN — The Daily Quest (15 min)

This is your **warm-up ritual**. It signals to your brain: "We are entering the dungeon now."

### Step 1: Shadow Extraction (5 min)

Open yesterday's Shadow Extraction. Re-read what you wrote. Then close it and rewrite it from memory.

```
[SYSTEM] Daily Quest — Shadow Extraction
───────────────────────────────────────────
  Yesterday's concept: [concept name]
  
  Write it from memory (no peeking):
  ┌─────────────────────────────────────┐
  │                                     │
  │                                     │
  │                                     │
  └─────────────────────────────────────┘
  
  Now compare with yesterday's version.
  Did you get it right? ☐ Yes  ☐ No
  If no, what was missing? _____________
```

### Step 2: Summon Shadows (5 min)

Open your Shadow Army. Pick **2 random Shadow Soldiers** (concepts you've extracted before). Read their descriptions. You're not re-studying — you're **reactivating** neural pathways.

```
[SYSTEM] Summoning Shadow Soldiers...
───────────────────────────────────────────
  Shadow 1: [concept name] — [rank]
  Your explanation: [brief]
  
  Shadow 2: [concept name] — [rank]
  Your explanation: [brief]
```

### Step 3: Dungeon Briefing (5 min)

Read today's mission objective. Know what you're building. Visualize the end state before you start.

```
[SYSTEM] Dungeon Briefing
───────────────────────────────────────────
  Today's Gate: [gate name]
  Today's Dungeon: [dungeon name]
  
  What I will build today:
  ┌─────────────────────────────────────┐
  │                                     │
  │                                     │
  └─────────────────────────────────────┘
```

## 🌤️ PHASE 2: DAY — The Dungeon Raid (2 × 90 min)

This is where the real work happens. Two 90-minute sprints with a break between them.

### Sprint 1: Build It (90 min)

```
═══════════════════════════════════════════
  SPRINT 1 — THE BUILD
═══════════════════════════════════════════

  ┌── RITUAL ──────────────────────────┐
  │  Close everything except:          │
  │  • This mission file               │
  │  • Your code editor                │
  │  • Terminal                        │
  │                                    │
  │  Phone: Silent. Face down.         │
  │  Browser: All non-essential tabs closed │
  │  Music: Lo-fi / instrumental (optional) │
  │                                    │
  │  You have 90 minutes.              │
  │  No switching. No checking. No scrolling. │
  └────────────────────────────────────┘

  ┌── THE PREDICTION ──────────────────┐
  │  Before you write ANY code:        │
  │  Write down what you expect to see │
  │  when you run it.                  │
  │                                    │
  │  "I think the output will be..."   │
  │  ┌──────────────────────────────┐  │
  │  │                              │  │
  │  │                              │  │
  │  └──────────────────────────────┘  │
  └────────────────────────────────────┘

  🚀 GO — 90:00 remaining
```

**Why the prediction matters:** When you predict before running, you're forced to form a mental model. If your prediction is wrong, that's where real learning happens — the gap between your model and reality.

### BREAK (20 min)

```
[SYSTEM] Sprint 1 complete.
[SYSTEM] Break mode activated.
───────────────────────────────────────────
  • Stand up. Walk away from the desk.
  • No screens. No phone scrolling.
  • Drink water. Look out the window.
  • Let your subconscious process.
  
  Your brain consolidates learning during breaks.
  Do NOT skip this.
───────────────────────────────────────────
```

### Sprint 2: The Twist (90 min)

```
═══════════════════════════════════════════
  SPRINT 2 — THE TWIST
═══════════════════════════════════════════

  ┌── THE TWIST ───────────────────────┐
  │  You've built the basic version.   │
  │  Now do the harder variant.        │
  │                                    │
  │  Each dungeon has a TWIST —        │
  │  a harder version of the same      │
  │  concept. This is where you        │
  │  truly learn.                      │
  └────────────────────────────────────┘

  ┌── BREAK IT ────────────────────────┐
  │  Intentionally break your code:    │
  │  1. Change something until it breaks │
  │  2. Read the error message         │
  │  3. Fix it                         │
  │                                    │
  │  This trains your Perception stat. │
  └────────────────────────────────────┘

  ┌── DEEP DIVE ───────────────────────┐
  │  NOW open the theory file.         │
  │  Read about the concept you just   │
  │  built with your hands.            │
  │                                    │
  │  It will make 10x more sense       │
  │  because you've already touched it.│
  └────────────────────────────────────┘

  🚀 GO — 90:00 remaining
```

```

  Write it in YOUR own words:
  ┌─────────────────────────────────────┐
  │                                     │
  │                                     │
  │                                     │

  you haven't mastered it yet.
```

### Step 2: Gate Check (3 min)

```
[SYSTEM] Gate Check
───────────────────────────────────────────
  ☐ Code runs without errors
  ☐ I typed every line (no copy-paste)
  ☐ I completed the twist
  ☐ I intentionally broke and fixed my code
  ☐ I can explain today's concept in 2 sentences
  
  ALL boxes checked? → Gate cleared. Move forward.
  Any box unchecked? → Repeat tomorrow before new content.
```

## The Weekly Schedule

Each week follows the same rotation:

```
DAY 1-5: New Content (Dungeon Raids)
  Standard protocol. Two sprints. New concepts.

DAY 6: RED GATE (Review Day)
  No new content. Active revision only.
  - Re-do 3 Shadow Extractions from memory
  - Re-type 1 key code example from the past week
  - Complete 1 Demon Castle challenge (from DEMON-CASTLE.md)
  - XP: +150

DAY 7: REST
  No training. Your streak continues.
  But if you WANT to train, there's bonus XP waiting.
  (Optional: Boss Battle catch-up or Demon Castle for +200 XP)
```

## The Golden Rules

```
┌──────────────────────────────────────────┐
│         THE 5 GOLDEN RULES               │
```
