## The Gate

Before you can build AI systems, you need to install the right mental models. Most engineers fail at AI not because they can't code — but because they approach it with the wrong mindset.

This Gate doesn't have code exercises. It has **mental training**. Each dungeon rewires how you think about AI engineering.

Your senior engineer brain already knows how to ship production systems. This Gate connects that knowledge to the AI world.
